ONG Solidária - pará
Cadastro de Voluntários
Descrição

O projeto ONG Solidária é uma página web onde voluntários, doadores e apoiadores podem se cadastrar. O site é responsivo, tem validação de formulários, modo escuro e alto contraste para acessibilidade, e navegação simples.

Funcionalidades

Cadastro de voluntários com validação de CPF, telefone, CEP e e-mail.

Formulário com mensagens de erro e sucesso.

Menu responsivo para celular (hambúrguer).

Modo escuro e alto contraste.

Navegação sem recarregar a página (SPA básica).

Design responsivo usando CSS Grid e Flexbox.

Tecnologias

HTML5

CSS3

JavaScript

Git/GitHub para controle de versão
